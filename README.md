## ğŸ—ï¸ Local Development

### Prerequisites

- Python 3.x
- pip

### Installation

1. Clone the repository:
```bash
git clone https://github.com/InfinityAlgo-Academy/Products.git
cd Products
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run local server:
```bash
mkdocs serve
```

4. Open browser:
```
http://127.0.0.1:8000
```

### Building the Site

```bash
mkdocs build
```

The site will be generated in the `site/` directory.
